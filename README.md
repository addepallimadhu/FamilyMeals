# FamilyMeals

Family Meals project is to enable a group of families or nuclear familes to have a common kitchen with flexibility to book meals per their preferred schedule. 

## Prompt used for initial version of code

Build a web application that serves the following requirements 
- A predefined set of users identified based on their phone numbers 
- The users have ability to book breakfast, lunch and dinner
- Booking can be for any period - could be for a day and even upto an year. Overlapping bookings shouldn't be allowed
- Group bookings are allowed but not for anyone outside set of users
- Booking to be done 24 hours in advance 
- More than 7 No shows the user will be restricted going forward. This no. of shows should be configurable so it can be changed easily
